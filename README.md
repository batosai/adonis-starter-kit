# Installation

```bash
npm init adonisjs@latest -- -K="batosai/adonis-starter-kit"
cd my-adonisjs-app
cp .env.example .env
node ace generate:key
node ace migration:run
node ace db:seed
```

## Developing

```bash
npm run dev
visite http://localhost:3333/admin

npm test
```
